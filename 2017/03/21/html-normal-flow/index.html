我对文档流的理解

  • html,normal-flow,文档流
技术

学前端的都知道文档流、脱离文档流等概念，英文Normal-Flow，翻译过来是文档流，我不知道为什么这么翻译，不过也不影响理解，也可以叫常规流、正常流，其实就是浏览器渲染页面的一种方式，就像上学时写作业一样，都是从上至下、从左至右，是一种默认的排列方式，块级元素独占一行，行内元素挨个排列，有正常的排列方式，就有不正常的（脱离文档流）。
