HTML标准模版

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no" />
    <title>HTML标准模版</title>

    <!-- css文件 -->
    <link rel="stylesheet" href="文件路径图" />
</head>

<body>

    <!-- js文件 -->
    <script src="文件路径"></script>
</body>
</html>

代码解析：

<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no" />

上述代码表示：强制文档的宽度与设备宽度一致，且不允许用户单击屏幕放大浏览。
