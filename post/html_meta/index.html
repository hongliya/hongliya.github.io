HTML：meta的一些用法

2017-03-27   #meta 

1.name=“keywords”

<meta name="keywords" content="test" />
// keywords 为文档定义一组关键字，搜索引擎遇到这些关键字的时候会用这些关键字对文档分类

2.name=“description”

<meta name="description" content="test" />
// 告诉搜索引擎网站的主要内容

3.http-equiv=“content-script-type”

<meta http-equiv="content-script-type" content="text/javascript" />
// W3C网页规范，指明页面中脚本的类型

4.http-equiv=“pragma”

<meta http-equiv="pragma" content="no-cache" />
// 禁止浏览器从本地计算机的缓存中访问页面内容，这样设定访问者将无法脱机浏览
// 脱机浏览：设置脱机查看网页后，可以在计算机未连接到Internet时阅读网页内容。

5.http-equiv=“cache-control”

<meta http-equiv="cache-control" content="no-cache, must-revalidate" />
// 防止html页面被缓存

6.http-equiv=“expires”

<meta http-equiv="expires" content="0" />
<meta http-equiv="expires" content="Mon,12 May 2001 00:20:00 GMT" />
// 设定网页的到期时间，一旦过期必须到服务器上重新调用，需要注意的是必须使用GMT时间格式

7.name=“viewport”

<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=0">
// 强制让文档的宽度和设备的宽度保持1:1，并且文档的最大宽度比例是1.0，且不允许用户点击屏幕放大浏览

8.name=“apple-mobile-web-app-capable”

<meta name="apple-mobile-web-app-capable" content="yes" />
// iphone设备中safari的私有meta标签，表示允许全屏模式浏览

9.name=“apple-mobile-web-app-status-bar-style”

<meta name="apple-mobile-web-app-status-bar-style" content="black" />
// iphone的私有标签，指定iphone中Safari顶端状态条的样式

10.name=“apple-mobile-web-app-capable”

<meta name="apple-mobile-web-app-capable" content="yes" />
// 网站开启对webAPP的支持

11.content=“telephone=no”

<meta name="format-detection" content="telephone=no" />
// 告诉设备忽略将页面中的数字识别为电话号码

12.content=“email=no”

<meta name="format-detection" content="email=no" />
// 告诉设备忽略自动识别页面中的邮件地址
// tips：ios不会自动识别页面中的邮件地址，android会自动识别页面中的邮件地址，并且会弹出框提示是否发送文件
