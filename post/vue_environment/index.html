Vue: Mac下Vue2.0环境配置

2018-03-19   #vue 

一、安装Homebrew

Homebrew是Mac系统下的软件包管理工具，用以简化Mac系统上的软件安装过程，相当于linux下的apt-get。

Homebrew的安装很简单，打开终端输入如下指令：

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

安装完成后就可以在命令行状态中使用brew命令了，可以用brew help命令查看帮助，一些常见命令：

brew -v               // 查看版本
brew install 软件名    // 安装软件
brew search 软件名     // 搜索软件
brew uninstall 软件名  // 卸载软件
brew upgrade 软件名    // 更新具体软件
brew list             // 显示已安装软件
brew info 软件名       // 查看软件信息
brew home 软件名       // 用浏览器打开官网网页查看软件信息
brew outdated         // 查看哪些已经安装的软件需要更新
brew update           // 更新所有软件，需要先安装git

二、安装Node.js

Node.js是一个JavaScript的运行环境，使得JavaScript代码能够在服务器端运行，跨平台。Node.js使得程序可以脱离Apache或者IIS，作为独立的服务器运行。

输入命令安装

brew install nodejs

三、安装淘宝镜像

根据国情，很多资源在局域网里访问不了，于是淘宝提供了一个资源站，把外网的一些东西下载好放到该资源站上，这样我们就可以下载，这就是镜像。也可以不装，直接用npm，看自己情况。

npm install -g cnpm --registry=https://registry.npm.taobao.org

四、安装Webpack

Webpack是一个打包工具，它能把各种资源（如JSX、coffee、less/sass、图片等）作为模块来使用和处理。浏览器无法直接解析.vue、.wxml、.wxss等自定义的组件，这时候webpack就可以把这些自定义组件打包成.js文件，以便浏览器解析。

cnpm install webpack -g

五、安装vue-cli

vue-cli是vue官方发布的项目脚手架，可以快速创建vue项目。

cnpm install vue-cli -g     // 全局安装vue-cli
npm view vue-cli            // 查看vue-cli版本号

六、开始一个项目

新建一个目录用来构建一个vue项目

mkdir vue                           // 新建vue目录
cd vue                              // 进入vue目录
vue init webpack-simple vue-demo    // 根据模版创建项目
// ...
// 设置项目名称、描述、作者
// 经过一些初始化设置后就项目就构建好了，然后根据提示进入项目并启动
// ...
cd vue-demo                         // 进入构建后的项目
npm install                         // 安装项目依赖（这一步耗时稍长）
npm run dev                         // 启动项目

输入完npm run dev后会自动启动浏览器，一切顺利的话就可以看到页面了。
